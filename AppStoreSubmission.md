### Notes for Reviewer
```
Leafwise diagnoses plants from photos using AI.

How it works:
1. The user takes or picks a plant photo.
2. The photo is resized to 1024px on-device before any network activity.
3. The resized image is sent over HTTPS to our Cloudflare Worker backend, authenticated with HMAC-SHA256.
4. The Worker forwards the image to OpenAI GPT-4o (Chat Completions API) using response_format: json_schema for structured output, then returns the result.
5. Results are persisted to the user's private iCloud database via SwiftData + CloudKit.

Privacy: No photos, images, or personal data are stored on our servers. The Cloudflare Worker has no database, KV store, or durable storage. No user account is required. The app uses identifierForVendor (a pseudonymous UUID that resets on full reinstall) solely for per-device rate-limiting on the backend.

Permissions:
- Camera (NSCameraUsageDescription): optional — only used if the user taps "Camera".
```

Leafwise uses only standard HTTPS (TLS) for all network requests. No custom cryptography beyond Apple's built-in frameworks (CryptoKit HMAC-SHA256 for request authentication).
